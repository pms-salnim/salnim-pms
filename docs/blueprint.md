# **App Name**: Salnim Pms

## Core Features:

- Property Overview: Dashboard overview of all managed properties.
- Occupancy Tracking: Track occupancy rates in all managed properties, with drill-down by date range.
- Maintenance Management: Manage maintenance requests, and assign them to relevant staff.
- Sentiment Analysis: AI-driven tool that analyzes guest feedback and identifies areas for improvement.
- User Management: User authentication and role-based access control.
- Financial Reporting: Financial tracking of income and expenses of each property.

## Style Guidelines:

- Primary color: A vibrant orange (#003166), evoking feelings of excitement and confidence.
- Background color: Very dark blue (#FFFFFF), for a sophisticated look.
- Accent color: Teal (#003166), which nicely complements the orange and contrasts well against the dark blue background.
- Body text: 'Inter', a sans-serif font for clean and modern readability.
- Headline text: 'Space Grotesk', a geometric sans-serif font suitable for short, impactful text.
- Use clear, outlined icons to represent property details and actions.
- Dashboard layout with clear data visualizations.
- Subtle transitions and animations for a smooth user experience.
