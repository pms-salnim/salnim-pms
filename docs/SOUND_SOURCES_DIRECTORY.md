## 🔧 Tools for Sound Processing

### Free Tools

**Audacity** - Best free audio editor
- **URL:** https://audacityteam.org
- **Features:** Edit, trim, normalize, export
- **Formats:** MP3, WAV, OGG, FLAC, etc.
- **Learn:** Built-in tutorials
- **Cost:** Free & open source

**FFmpeg** - Command line converter
- **Installation:** Homebrew (`brew install ffmpeg`)
- **Convert WAV to MP3:** `ffmpeg -i sound.wav sound.mp3`
- **Normalize audio:** `ffmpeg -i sound.mp3 -af loudnorm=I=-23:TP=-1.5:LRA=11 normalized.mp3`
- **Trim to 2 seconds:** `ffmpeg -i sound.mp3 -t 2 trimmed.mp3`

**VLC Player** - Works as converter
- **Free:** Open media → Export audio
- **Formats:** Convert between many formats

**MP3Decrease** - Online compressor
- **URL:** https://mp3decrease.com
- **Use:** Reduce file size
- **Max:** 100MB per file

### Premium Tools

**Adobe Audition** - Professional audio editing
- Part of Creative Cloud
- Advanced editing & effects

**Reaper** - Digital Audio Workstation
- Affordable ($60 one-time)
- Professional features

## 📝 License Best Practices

**Always Check:**
1. License type (Creative Commons, Public Domain, etc.)
2. Attribution requirements (do you need to credit?)
3. Commercial use (allowed or not?)
4. Modification rights (can you edit?)
5. Derivative works (can you use in apps?)

**For Web Applications:**
- ✅ Creative Commons (CC0, CC-BY if you attribute)
- ✅ Public Domain
- ✅ Paid licenses (AudioJungle, Epidemic, etc.)
- ⚠️ CC-BY-NC (non-commercial only)
- ❌ Rights-managed (need permissions)

## 🆘 Troubleshooting

### Can't Find the Perfect Sound?
1. Try different search terms
2. Browse similar sounds
3. Check different platforms
4. Consider editing found sound
5. Use built-in presets instead

### File Too Large?
1. Use MP3 format (smaller than WAV)
2. Download lower bitrate (128 kbps)
3. Use MP3Decrease.com to compress
4. Trim silence with Audacity

### Audio Quality Issues?
1. Use 128-192 kbps MP3 (good balance)
2. Check WAV version if available
3. Normalize audio in Audacity
4. Try different source
